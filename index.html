Comment jouer ?

  • Déplace les éléments entre les plateaux pour modifier l'équation
  • Utilise les opérations mathématiques pour simplifier l'équation
  • L'objectif est d'isoler x d'un côté du plateau
  • Rappelle-toi : toute opération faite d'un côté doit être faite de l'autre !

Concept de Base

Une équation est comme une balance : les deux côtés doivent toujours être égaux !

Pour résoudre une équation du type x + 3 = 8 :

  1. Enlève +3 du côté gauche en ajoutant -3 des deux côtés : x + 3 - 3 = 8 - 3
  2. Simplifie : x = 5

Dans ce jeu, tu peux :

  • Déplacer les nombres d'un côté à l'autre (ils changent de signe)
  • Ajouter/soustraire le même nombre des deux côtés
  • Multiplier/diviser les deux côtés par le même nombre
